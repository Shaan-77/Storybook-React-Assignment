# Storybook React Project

The Storybook React Project is a component-driven front-end application built with **React**, **TypeScript**, and **Vite**, showcasing reusable UI components such as a generic DataTable with sorting and selection features. It integrates **Storybook** for isolated development and documentation, and Vitest for testing, emphasizing scalability, maintainability, and type safety in modern web development.

---

## üöÄ Features

- **Component-Driven Development** ‚Äì Build, preview, and document components in isolation with Storybook.  
- **Reusable UI Components** ‚Äì Includes DataTable, InputField, Button, and more with customizable props.  
- **Advanced DataTable** ‚Äì Sorting, single/multi-row selection, custom rendering, and loading/empty states.  
- **Type Safety** ‚Äì Strongly typed with TypeScript and generics for safer, scalable code.  
- **Scalable Structure** ‚Äì Clean folder organization for maintainability and growth.  
- **Testing Integration** ‚Äì Unit and integration testing with Vitest for reliability.  
- **Optimized Performance** ‚Äì Powered by Vite for lightning-fast builds and HMR.  
- **Customizable Styling** ‚Äì Flexible design with easy theming and style overrides.

## üì¶ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone [<repo-url>]
   cd Storybook-React
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
   Access the app at: [http://localhost:5173](http://localhost:5173)

4. **Run Storybook**
   ```bash
   npm run storybook
   ```
   Access Storybook at: [http://localhost:6006](http://localhost:6006)

5. **Build for production**
   ```bash
   npm run build
   ```

6. **Preview production build**
   ```bash
   npm run preview
   ```

7. **Run tests**
   ```bash
   npm run test
   ```

- **`src/SampleData/`** ‚Üí Mock/sample dataset for demo and testing  
- **`storybook-static/`** ‚Üí Pre-built static Storybook distribution
